# Restaurant Application

##  Project Description

The Restaurant Application is a Python-based console program designed to manage restaurant operations efficiently. The system allows users to view menus, place orders, calculate bills, and streamline customer interactions through a simple and user-friendly interface.

This project demonstrates the use of core Python concepts such as functions, lists, dictionaries, loops, and conditional statements.

---

##  Features

* Display restaurant menu
* Add items to order
* Calculate total bill automatically
* Generate order summary
* Simple and interactive console interface

---

##  Technologies Used

* Python 3
* Basic Data Structures (Lists, Dictionaries)
* Functions
* Conditional Statements & Loops

##  How to Run the Project

### Step 1: Install Python (if not installed)

Download and install Python from the official website.

### Step 2: Run the Application

```bash
python restaurant_app.py
```

---

## System Workflow

1. Display the menu with item names and prices.
2. User selects items and quantity.
3. Items are added to the cart.
4. The system calculates the total bill.
5. Order summary is displayed.

---

## Concepts Used

* **Functions** – To organize code into reusable blocks.
* **Dictionaries** – To store menu items and prices.
* **Loops** – To handle multiple orders.
* **Conditionals** – To validate user input.
